# RhemaFlow UI Polish Design

**Goal:** Elevate the app's visual quality to a premium/luxury feel with glass effects, gold accents, and subtle depth â€” without changing functionality.

**Direction:** Keep the dark/fire/spiritual brand, add frosted glass treatments, gold shimmer accents, refined typography, and smoother interactive states.

## Sections

### 1. App Icon & Splash Screen
- Flame on deep purple-to-black gradient with gold halo ring
- Splash: centered "RhemaFlow" wordmark in Cinzel (gold on void black) with radial glow

### 2. Tab Bar & Navigation
- Frosted glass tab bar (BlurView) instead of solid black
- Active tab: gold glowing dot indicator beneath icon
- Larger icons (26px), tighter label spacing
- Gold accent line beneath Home header
- User icon: gold ring border when signed in
- Back chevron: small circular glass pill background

### 3. Home Screen
- "YOUR VOICE" gold gradient text instead of flat fireOrange
- Category pills: subtle inner glow on selected, faint border on unselected
- Mood preset cards: glass border effect, more padding, rounder corners
- Custom input: glass-style blur background, gold gradient submit button

### 4. Declaration Card
- Refined gradient overlay stops
- Gold drop shadow on declaration text
- Scripture blur intensity increased (frosted glass)
- Play button: gold border ring idle, pulsing glow when playing
- All control buttons: glass circular backgrounds

### 5. Saved Screen
- Left-side category-color accent bar on list cards
- Increased card padding and soft elevation
- Detail header: gold accent line

### 6. Settings Screen
- Avatar: gradient border (purple to gold) with subtle glow
- Setting row icons: small tinted circle backgrounds

### 7. Loading Overlay
- Gold flame icon with smooth pulse
- "IGNITING THE WORD..." in Cinzel with letter-spacing
